Usage:
  syk4y init [--repo-root DIR] [options] <artifact...>

Options:
  -u, --upload-dir DIR   Upload root (default: kaggle_upload under repo root)
  -w, --wheelhouse FILE  Wheelhouse archive filename/path (default: wheelhouse.zip)
  -h, --help             Show this help

Artifacts:
  Any folder/file path under repo root, or absolute path.
  For non-'wheelhouse' artifacts, path must already exist (file, directory, or symlink).
  Special case: artifact name exactly 'wheelhouse' triggers wheelhouse build flow.

Wheelhouse Artifact Notes:
  - When 'wheelhouse' is included, syk4y builds/updates wheelhouse.zip from the current Python environment.
  - This step does not require Kaggle login and does not upload to Kaggle.

Install From Wheelhouse:
  - Extract wheelhouse.zip first (example: unzip wheelhouse.zip -d wheelhouse)
  - Prefer uv (faster):
      uv pip install --offline --no-index --find-links /path/to/wheelhouse <package-or-requirements>
  - Or pip:
      pip install --no-index --find-links /path/to/wheelhouse <package-or-requirements>
