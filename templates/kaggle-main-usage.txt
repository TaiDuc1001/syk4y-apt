Usage:
  syk4y kaggle [--repo-root DIR] <subcommand> [options]

Subcommands:
  login    Configure Kaggle credentials (~/.kaggle/kaggle.json)
  upload   Upload changed artifacts to Kaggle (requires Kaggle credentials)
  account  Manage Kaggle account pool (add/list/remove/enable/disable/test)
  run      Multi-account notebook runner (start/status/list/pull/resume)

Examples:
  syk4y kaggle login
  syk4y kaggle upload --force
  syk4y kaggle upload datasets wheelhouse
  syk4y kaggle account add --id main --username user1 --key xxxxx
  syk4y kaggle run start --notebook experiment.ipynb
